# 🌾 乡野直达 · 助农农产品交易平台

一个纯前端、基于浏览器本地存储的助农电商演示系统，包含用户购物、商家后台和管理员监控三大模块。

## ✨ 主要功能

### 🛒 用户端（首页 & 购物车）
- 浏览商品列表（支持横向拖拽滚动 + 惯性滑动）
- 搜索商品（按名称或产地）
- 加入购物车、修改数量、删除商品
- 下单结算（订单自动保存到本地）

### 🛍️ 商家端（商家中心）
- 登录密码：`merchant123`
- 商品管理：增删改商品（支持单独修改某一字段）
- 消息回复：查看用户咨询并回复（回复内容会显示在用户聊天窗口）
- 消息撤回：自己发送的消息可以在 2 分钟内撤回

### 📊 管理员端（管理后台）
- 登录密码：`admin123`
- 数据看板：总订单数、总销售额、商品销量排行
- 订单管理：查看所有订单、删除订单、自动清理两个月以上的旧订单
- 聊天监控：查看所有用户与商家的聊天记录，可删除任意消息

### 💬 即时聊天
- 用户点击首页右下角“联系商家”浮动按钮，可发送消息
- 商家在后台实时查看并回复
- 用户和商家各自只能删除自己发送且未超过 2 分钟的消息
